== Description ==

Displays CATEGORIES of your **[PrestaShop](http://www.prestashop.com "an opensource ecommerce solution")** ecommerce website on your blog.
Displays "home categories" (level depth = 1) on your blog.

You need to install the **PrestaShop module for WordPress categories** on your ecommerce website to use this plugin : [http://prestashop.joelgaujard.info/product.php?id_product=17](http://prestashop.joelgaujard.info/product.php?id_product=17 "PrestaShop module for WordPress")

Some functionalities :

* configure the URL of your PrestaShop ecommerce website
* creation of one widget to displays categories
* specify if you want to display image category


== Installation ==

1. Upload the folder `wp-prestashop-categories` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Buy and install prestashop module : wordpress categories

== Frequently Asked Questions ==

= What I have to put on my PrestaShop website ? =

You have to download the [module wordpress categories for PrestaShop](http://prestashop.joelgaujard.info/product.php?id_product=17 "a prestashop module to displays wordpress categories") created by the same author of this module.
